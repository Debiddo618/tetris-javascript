Game Rules

  1. Tetrominoes: The game involves different shapes called tetrominoes falling from the top of the game board to the bottom. Each tetromino is made up of four square blocks arranged in various configurations.
  2. Movement: Players can move the falling tetrominoes left or right using the arrow keys. They can also rotate the tetrominoes clockwise using the up arrow key to fit them into available spaces. Use the down arrow key to drop the pieces faster.

Objective

  1. The goal is to complete horizontal lines across the game board. When a line is completed without any gaps, it disappears, and any blocks above it move down to fill the space.
  2. Players earn points for each completed line. 100 points per line.

Lose Conditions

  1. The game ends when the stack of tetrominoes reaches the top of the game board, preventing new pieces from entering.
